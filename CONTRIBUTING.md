# このリポジトリへの貢献方法

バグ報告・提案・質問は Issue を利用してください。

Pull Request を歓迎します。
大きな変更を行う場合は、先に Issue で相談してください。

## コーディング方針

- Pythonのバージョンは 3.14t を想定しています。詳しくは、 pyproject.toml を参照してください。
- Ruff を使用してコードスタイルをチェックしています。
- コメント・ドキュメントは日本語で記述してください。

## データについて

個人情報や未匿名化データを含む Pull Request は受け付けません。

## 生成AI

生成AIを使用してコードやドキュメントを作成することは問題ありませんが、生成された内容を必ず人間が確認し、必要に応じて修正してください。
AI支援によるコード生成は禁止しませんが、内容の確認責任は投稿者にあります。

## セキュリティ問題の報告

SECURITY.md を参照してください。
