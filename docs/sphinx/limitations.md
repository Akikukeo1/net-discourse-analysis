# 本プロジェクトの制限事項

このドキュメントは、プロジェクトの実行時における既知の制限事項をまとめたものです。

## 1. ハードウェア並列処理の制限

- CPUコア数やメモリ容量に応じて、最適なワーカー数が自動的に決定されますが、実行環境の負荷状況によってはパフォーマンスが低下する可能性があります。

## 2. YouTube API / スクレイピングの制限

- YouTubeコメントの取得は外部ライブラリ（`youtube-comment-downloader`）に依存しているため、YouTube側の仕様変更やレートリミット（アクセス制限）によるエラーが発生する可能性があります。
